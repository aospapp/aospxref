# aospxref
Deploy your own AOSPXRef

## Usage

Clone aosp to /data/aospxref/src

You can remove `.repo`, `prebuilts` to reduce size

```
/data/aospxref/src
├── android-10.0.0_r1
├── android-11.0.0_r1
├── android-12.0.0_r1
├── android-12.1.0_r1
├── android-13.0.0_r1
├── android-5.0.0_r1
├── android-5.1.0_r1
├── android-6.0.0_r1
├── android-7.0.0_r1
├── android-7.1.0_r1
├── android-8.0.0_r1
├── android-8.1.0_r1
└── android-9.0.0_r1
```

Edit version.txt to your local versions

then run gen.py to generate docker-compose.yml
